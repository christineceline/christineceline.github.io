The Portfolio

So why not start with having a closer look at the making of this portfolio itself!
The structure is based on the JavaScript plugin fullPage.js, changed it to suit my requirements. The layout is built using CSS grid, to enable easy viewing on both bigger and smaller screens.
I used JS and CSS to build modals, to demonstrate projects 2 and 4.
Finally, despite the lack of a back-end to this website, the contact form at the bottom of the site is fully functional and powered by getform.io.
The site is hosted on GitHub.

Overall this portfolio is an excellent example of how free resources can be used to make a fully functional, unique, and personal website.
